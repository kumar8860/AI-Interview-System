## ⚙️ Project Setup (Follow Carefully)

### 🔽 Step 1: Clone Project

git clone https://github.com/kumar8860/AI-Interview-System.git

---

### 🔽 Step 2: Backend Setup

cd backend
python -m pip install -r requirements.txt
python app.py

---

### 🔽 Step 3: Frontend Setup

cd frontend
npm install
npm start

---

### 🔽 Step 4: ML Setup

cd ml_models
python -m pip install -r requirements.txt

---

## 🌐 Run Project

* Backend → http://127.0.0.1:5000
* Frontend → http://localhost:3000
