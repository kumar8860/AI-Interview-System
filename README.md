# AI Interview System

## 👥 Team Roles

* Backend & Integration → Member 1
* Frontend (UI/UX) → Member 2
* AI/ML → Member 3

---

## 📁 Project Structure

frontend/ → UI (React)
backend/ → API (Flask)
ml_models/ → AI logic

---

## ⚙️ Setup Instructions

### 1. Clone Project

git clone https://github.com/kumar8860/AI-Interview-System.git

---

### 2. Backend Setup

cd backend
python -m pip install -r requirements.txt
python app.py

---

### 3. Frontend Setup

cd frontend
npm install
npm start

---

### 4. ML Setup

cd ml_models
python -m pip install -r requirements.txt

---

## 🌐 Run Project

Backend → http://127.0.0.1:5000
Frontend → http://localhost:3000
